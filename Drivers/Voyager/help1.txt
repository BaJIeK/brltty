---BRLTTY Voyager 70 Help Screen---

Key Names:
  Front Panel Keys:
    Leftmost key is named A; second key from left is named B.
    Rightmost key is named D; second key from right is named C.
    Round extension to left of central keypad is named RL; right is RR.
    Up position of central keypad is named UP; down is DOWN.
  Round keys on top are used as a braille keyboard.
  Cursor routing keys:
    CRt1 to CRt66: routing keys over text cells.
    CRs1 to CRs4: routing keys over status cells.
    CRa1 to CRa70: all routing keys counting from left extremity.

Basic Navigation
A / D: Move backward/forward
B / C: Move up/down
A+B / A+C: Goto top-left / bottom-left
RR: Goto cursor
RL: Cursor tracking toggle
UP / DOWN: Move cursor up/down (arrow keys)

Modes and Functions
CRs1: Help screen (toggle)
CRs1+CRs2: Learn mode (key describer) (toggle)
RL+RR: Freeze screen (toggle)
RL+UP: Show attributes (toggle)
RR+UP: Show position and status info (toggle)
CRs2: Preferences menu (and again to exit)
CRa#+UP: Switch to virtual console #
CRa1+CRa2+RL / RR: Switch to previous/next virtual console
A+D+CRa1: Six dots mode (toggle)

Cursor Routing
CRt#: Route cursor to cell
CRs4: Route cursor to current line

Cut and Paste
CRt#+D: Mark beginning of region to cut
CRt#+A: Mark bottom-right of rectangular region and cut
CRt# + CRt(#+1) + D: Mark beginning of cut region for append
CRt# + CRt(#-1) + A: Mark end of linear region and cut
CRtx + CRt(x+2) + CRty: Cut text from x to y
CRt2+CRt3: Paste cut text

Fancy Navigation
RL+B / RL+C: Previous/next line with different attributes
RR+B / RR+C: Previous/next different line
RR+A / RR+D: Previous/next non-blank window
CRs3: Go back to previous reading location (undo cursor tracking motion).
CRt1+CRt2 / CRt65+CRt66: Move window left/right one character
CRa#+RL: Remember current position as mark #
CRa#+RR: Goto mark #
CRt#+DOWN: Move right # cells
CRt#+B / C: Move to previous/next line indented no more than #
CRt1+CRt2+B / C: Move to previous/next paragraph (blank line separation)
CRt1+CRt3+B / C: Search screen backward/forward for cut text
CRt2+CRt3+B / C: Previous/next prompt (same prompt as current line)

Typing in Braille
Chord-1478: Toggle braille input on/off
B+C: Space (spacebar)
Chord-46: Return
Chord-2345: Tab
Chord-12: Backspace
Chord-246: Escape
Chord-145: Delete
Chord-7: Left arrow
Chord-8: Right arrow
