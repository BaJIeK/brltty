---BRLTTY MiniBraille Key Bindings help---

This help: F1+F1+F1

Used keys from left:
  F1 F2 Left Up C Down Right

Modes:
  Basic (activated after start)
  F1 mode (use F1 from basic mode)
  F2 mode (use F2 from basic mode)
  Management mode (F1+F1)
  Clock mode (F1+F1+Right)

Movement: (from basic mode)
  Move cursor 20 chars left: Left
  Move cursor 20 chars right: Right
  Move cursor line up: Up
  Move cursor line down: Down
  Begin of line: F1+Left
  End of line: F1+Right
  Top of screen: F1+Up
  Bottom of screen: F1+Down

Basic mode:
F1: goto F1 mode
F2: goto F2 mode
Left: 20 chars left
Up: line up
C: goto cursor
Down: line down
Right: 20 chars right

F1 mode:
F1: goto management mode
F2: toggle cursor visibility
Left: begin of line
Up: top of screen (leftmost)
C: toggle blinking of capitals
Down: bottom of screen (leftmost)
Right: end of line

F2 mode:
F1: toggle size of cursor
F2: toggle skipping of identical line
Left: show info
Up: freeze screen
C: track cursor
Down: toggle attribute/text
Right: toggle 6/8 dots display

Management mode:
F1: help
F2: show info
Left: restart driver
Up: enter preferences menu
C: reset to preferences
Down: save preferences
Right: goto CLOCK mode

Clock mode:
The display shows current date and time in this mode.
Use any key to leave this mode.

End of help.
